## 比赛简介 ##
该比赛是香港上海汇丰银行与上海纽约大学共同举办的建模大赛，比赛内容为银行提供近万名客户数据并要求参赛者在36小时内对此进行建模并预测下季度购买其产品的具体客户名单，正确率高者为胜。
